# Personal Portfolio Website

A simple and responsive personal portfolio website built using **HTML** and **CSS**. This website contains two main pages: **Home** and **Contact**. It is designed to present personal information, showcase a clean web layout, and provide an easy way for visitors to get in touch.

## Technologies Used

* HTML5
* CSS3

## Pages

* **Home Page** – Displays introduction and portfolio overview
* **Contact Page** – Contains contact information or contact form

## Purpose

This project was created to practice and improve frontend web development skills using HTML and CSS while building a personal portfolio website.

Thank you for visiting my portfolio website!
